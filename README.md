## 爬虫相关

- 由于爬虫采用 nodeJs 进行编写
- 首先确保您已安装 nodeJS
- 运行前请在 spider 目录下执行 `npm install`
- 确保已全局安装 typescript 依赖包
- 而后在执行 src 目录下的 main.ts 文件即可（IDE 为 vscode 建议安装 Code Runner
  插件）
- 报错 1： 请执行完 ts 命令后再次执行所生成的 js 文件
- 报错 2： 请前往阿里 v 任务首页进行登录，并复制 cookies 替换 cookies 中的
  taobaoCookies
